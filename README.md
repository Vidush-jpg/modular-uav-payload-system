# Modular UAV Payload System

A modular payload architecture for FPV UAVs featuring a standardized backplane,
plug-and-play payload identification, and companion-computer integration.

The working project is under
`Modular_UAV_Payload_Starter/modular_uav_payload_repo_starter`.

- Arduino Uno R4 experiments live under `prototypes/arduino_uno_r4`.
- The main Raspberry Pi implementation belongs under `raspberry_pi`.
- Shared specifications and architecture remain under `docs`.
